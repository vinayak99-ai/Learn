# Top GitHub Repositories for LLM-Powered AI Workflows & Agents

A curated list of the most popular GitHub repositories leveraging Large Language Models (LLMs) for AI workflows, autonomous agents, and agentic automation, sorted by GitHub stars.

---

## Workflow Automation & Agent Frameworks

| Repository | Stars | Description |
|------------|-------|-------------|
| <a href="https://github.com/n8n-io/n8n">n8n</a> | 160k+ | Open-source workflow automation platform with native AI agent features via LLM integrations (OpenAI, Local LLMs using LangChain). Supports 400+ integrations for custom agentic automations. |
| <a href="https://github.com/logspace-ai/langflow">Langflow</a> | 140k+ | Visual builder for LLM pipelines, agent orchestration, RAG systems, and multi-agent chat. Drag-and-drop interface built on LangChain. |
| <a href="https://github.com/Significant-Gravitas/Auto-GPT">Auto-GPT</a> | 160k+ | Pioneer in fully autonomous agent frameworks. Given a goal, the agent executes multi-step plans using LLMs for research, content creation, software development. |
| <a href="https://github.com/langchain-ai/langchain">LangChain</a> | 100k+ | Gold-standard Python library for building LLM-powered applications and agentic workflows. Simplifies prompt engineering, memory, RAG, tool use, and agent loops. |
| <a href="https://github.com/crewAIInc/crewAI">CrewAI</a> | 25k+ | Framework for multi-role, collaborative, and autonomous LLM agents to tackle complex tasks by working together. |
| <a href="https://github.com/langgenius/dify">Dify</a> | 60k+ | Rapid RAG system and bot builder to create intelligent, retrieval-augmented agents. All-in-one ingestion, search, orchestration, and deployment. |

## Additional Notable Repositories

- <a href="https://github.com/ollama/ollama">ollama</a> - Get up and running with large language models locally
- <a href="https://github.com/run-llama/llama_index">LlamaIndex</a> - Data framework for LLM applications

## How to Use This List

1. **For building AI agents**: Start with LangChain, CrewAI, or Auto-GPT
2. **For workflow automation**: Try n8n or Langflow for visual building
3. **For RAG applications**: Check out Dify, LlamaIndex, or Haystack
4. **For running local LLMs**: Use Ollama or GPT4All
5. **For learning**: Explore the Prompt Engineering Guide and Awesome-LLM-Apps
